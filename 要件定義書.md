# 胸部X線画像診断AIシステム 要件定義書

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | 胸部X線画像多疾患診断AIシステム |
| 目的 | 胸部X線画像から14種類の疾患を自動検出する深層学習モデルの構築 |
| データセット | NIH ChestX-ray14（112,120枚） |
| 開発環境 | Ubuntu / NVIDIA GTX 1080 Ti |
| 作成日 | 2026-04-17 |

---

## 2. 背景と目的

胸部X線検査は最も一般的な医療画像検査であり、肺炎・肺がん・心肥大など多くの疾患の診断に用いられる。放射線科医の読影業務の負担軽減および見落とし防止を目的として、深層学習による自動診断支援システムを構築する。

## 5. 機能要件

### 5.1 学習機能
- DenseNet-121をImageNetで事前学習済みの重みで初期化（転移学習）
- 多ラベル分類（1枚の画像に複数の疾患が共存可能）
- クラス不均衡対策（pos_weightによる損失関数の重み付け）
- 混合精度学習（FP16）によるGPUメモリ効率化
- CosineAnnealingによる学習率スケジューリング

### 5.2 データ拡張
- ランダム水平反転
- ±15度のランダム回転
- 輝度・コントラスト変動（ColorJitter）
- ランダムクロップ（256→224px）

### 5.3 評価機能
- AUC-ROC（疾患ごと・平均）
- 5エポックごとの検証評価
- ベストモデルの自動保存
- TensorBoardによる学習曲線の可視化

### 5.4 可視化機能
- Grad-CAMによる診断根拠のヒートマップ表示

---

## 6. 非機能要件

| 項目 | 要件 |
|------|------|
| 精度目標 | Mean AUC ≥ 0.80（論文ベースライン: 0.841） |
| 学習時間 | 30エポックで24時間以内 |
| 推論速度 | 1枚あたり100ms以内 |
| 再現性 | random_seed固定による再現可能な学習 |
